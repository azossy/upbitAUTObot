## 4. 백엔드 인증 라우트·스키마 (코드 검증)

백엔드에서 **pytest/venv 미설정** 환경이라 런타임 import 테스트는 생략. 대신 코드 기준으로 엔드포인트·스키마 존재 여부 확인.

| 엔드포인트 | 메서드 | 응답 모델 | 용도 |
|------------|--------|-----------|------|
| `/api/v1/auth/google` | POST | OAuthLoginResponse | 구글 로그인 (id_token). need_register 시 email/name 반환 |
| `/api/v1/auth/kakao` | POST | OAuthLoginResponse | 카카오 로그인 (access_token). need_register 시 email/name 반환 |
| `/api/v1/auth/complete-google-register` | POST | TokenResponse | 구글 OAuth 회원가입 완료 (id_token + nickname) |
| `/api/v1/auth/complete-kakao-register` | POST | TokenResponse | 카카오 OAuth 회원가입 완료 (access_token + nickname) |

- **OAuthLoginResponse**: `need_register`, `access_token`, `token_type`, `expires_in`, `user`, `email`, `name`.
- **CompleteGoogleRegisterRequest**: `id_token`, `nickname`.
- **CompleteKakaoRegisterRequest**: `access_token`, `nickname`.
- `_fetch_kakao_user`는 `async def`로 정의되어 비동기 호출 정합성 유지.

## 5. 수동·통합 테스트 권장 (실기/에뮬레이터)

다음은 **실제 기기 또는 에뮬레이터**에서 권장하는 수동 테스트 절차입니다. (실행 여부는 환경에 따라 별도 진행.)

### 5.1 사전 조건

- 백엔드 서버 기동: `cd backend && uvicorn main:app --host 0.0.0.0 --port 8000`
- 앱 설정: API 서버 주소가 해당 백엔드 주소로 설정됨.
- **구글**: Google Cloud Console에서 Web 클라이언트 ID 발급 후, 앱 `OAuthConfig.googleWebClientId`와 백엔드 `.env`의 `GOOGLE_CLIENT_ID`에 동일 값 설정.
- **카카오**: Kakao Developers에서 네이티브 앱 키·REST API 키 발급 후, 앱 `OAuthConfig.kakaoNativeAppKey`와 백엔드 `.env`의 `KAKAO_REST_API_KEY` 설정.

### 5.2 구글 로그인 플로우

1. 로그인 화면에서 「구글 계정으로 로그인」 탭.
2. **미가입 구글 계정** 사용 시:  
   - 서버가 `need_register: true` + email/name 반환 → 회원가입 화면으로 자동 이동.  
   - 이메일(읽기 전용), 닉네임 입력 후 「가입 완료」 → 로그인 완료 후 홈 이동.
3. **이미 가입된 구글 계정** 사용 시:  
   - 서버가 JWT 반환 → 바로 홈(자동 로그인).

### 5.3 카카오 로그인 플로우

1. 로그인 화면에서 「카카오 계정으로 로그인」 탭.
2. **미가입 카카오 계정** 사용 시:  
   - 서버가 `need_register: true` + email/name 반환 → 회원가입 화면으로 자동 이동.  
   - 이메일(읽기 전용), 닉네임 입력 후 「가입 완료」 → 로그인 완료 후 홈 이동.
3. **이미 가입된 카카오 계정** 사용 시:  
   - 서버가 JWT 반환 → 바로 홈(자동 로그인).

### 5.4 API 직접 호출 예시 (curl)

- **구글 로그인** (실제 id_token으로 교체 필요):
  ```bash
  curl -s -X POST http://127.0.0.1:8000/api/v1/auth/google -H "Content-Type: application/json" -d "{\"id_token\":\"YOUR_GOOGLE_ID_TOKEN\"}"
  ```
- **카카오 로그인** (실제 access_token으로 교체 필요):
  ```bash
  curl -s -X POST http://127.0.0.1:8000/api/v1/auth/kakao -H "Content-Type: application/json" -d "{\"access_token\":\"YOUR_KAKAO_ACCESS_TOKEN\"}"
  ```
- **구글 OAuth 회원가입 완료**:
  ```bash
  curl -s -X POST http://127.0.0.1:8000/api/v1/auth/complete-google-register -H "Content-Type: application/json" -d "{\"id_token\":\"...\",\"nickname\":\"테스트닉네임\"}"
  ```
