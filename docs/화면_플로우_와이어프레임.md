## 4. 와이어프레임 개요

### 4.1 로그인 화면
```
┌─────────────────────────────┐
│        [앱 로고]             │
│                              │
│  ┌──────────────────────┐   │
│  │ 이메일                │   │
│  └──────────────────────┘   │
│  ┌──────────────────────┐   │
│  │ 비밀번호              │   │
│  └──────────────────────┘   │
│                              │
│  ┌──────────────────────┐   │
│  │      로그인           │   │  ← Contained 버튼
```

## 5. One UI 적용 사항

| 항목 | 적용 |
|------|------|
| 구조 | 보는 영역(상단) / 인터랙션 영역(하단) 구분 |
| 여백 | 좌우 24dp 이상 |
| 버튼 | 주요 액션: Contained, 툴바: Flat |
| 다이얼로그 | 화면 하단에서 위로 |
| 다크 모드 | 지원 |
| 하단 네비 | 4개, 아이콘+텍스트 |
