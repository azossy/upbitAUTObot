## 1. 개요

### 1.1 제품 목적
업비트 거래소 API를 활용하여 **자동매매**를 수행하고, 사용자가 스마트폰 앱으로 제어·모니터링할 수 있는 시스템.

### 1.2 목표 사용자
- 소규모 다중 사용자 (가족, 지인 등)
- 개인 투자자

### 1.3 핵심 가치
- 24시간 자동매매 (서버 기반)
- 모바일 앱으로 간편한 제어·모니터링
- 매수/매도/손절 실시간 푸시 알림

---

## 2. 아키텍처

### 2.1 하이브리드 구조

```
[Flutter 앱 - Android]  ←→  [Python FastAPI 서버]  ←→  [업비트 API]
        │                            │
        │                            ├── SQLite DB
        │                            ├── 업비트 REST/WebSocket
        │                            └── FCM / 텔레그램
        └── FCM 푸시 수신
```

### 2.2 기술 스택

| 구분 | 기술 |
|------|------|
| 서버 | Python + FastAPI |
| DB | SQLite |
| 클라이언트 | Flutter (Android 우선) |
| 서버 실행 환경 | 집 PC 또는 Jetson Nano 8GB |
| 푸시 | FCM + 텔레그램 |

## 3. 기능 요구사항

### 3.1 인증 (Auth)
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| AUTH-01 | 회원가입 (이메일, 비밀번호, 닉네임) | 필수 |
| AUTH-02 | 로그인 (JWT Access/Refresh Token) | 필수 |
| AUTH-03 | 로그아웃 | 필수 |
| AUTH-04 | 비밀번호 변경 | 필수 |
| AUTH-05 | 계정 잠금 (로그인 5회 실패 시 15분) | 권장 |

### 3.2 업비트 연동
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| UPBIT-01 | API Key 등록 (Access Key, Secret Key) | 필수 |
| UPBIT-02 | API Key 암호화 저장 (AES-256) | 필수 |
| UPBIT-03 | 잔고 조회 | 필수 |
| UPBIT-04 | 주문 생성/취소 | 필수 |
| UPBIT-05 | 실시간 시세 (WebSocket) | 필수 |

### 3.3 봇 제어
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| BOT-01 | 봇 시작 | 필수 |
| BOT-02 | 봇 정지 | 필수 |
| BOT-03 | 봇 상태 조회 (실행중/중지/에러) | 필수 |
| BOT-04 | 시장 모드 표시 (상승/횡보/하락) | 필수 |

### 3.4 대시보드
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| DASH-01 | 잔고 표시 | 필수 |
| DASH-02 | 일일/주간/누적 수익률 | 필수 |
| DASH-03 | 승률 표시 | 필수 |
| DASH-04 | 보유 포지션 목록 (코인, 수량, 수익률) | 필수 |

### 3.5 거래 내역
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| TRADE-01 | 거래 내역 목록 (매수/매도) | 필수 |
| TRADE-02 | 기간별 필터 (7/30/90일) | 필수 |
| TRADE-03 | 코인별 필터 | 권장 |

### 3.6 설정
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| SET-01 | 투자 비율 설정 | 필수 |
| SET-02 | 손절/익절 기준 설정 | 필수 |
| SET-03 | API Key 등록/삭제 | 필수 |
| SET-04 | 텔레그램 Chat ID 등록 | 필수 |

### 3.7 푸시 알림
| ID | 요구사항 | 우선순위 |
|----|----------|----------|
| PUSH-01 | FCM: 매수/매도 체결 알림 | 필수 |
| PUSH-02 | FCM: 손절 알림 | 필수 |
| PUSH-03 | 텔레그램: 매수/매도/손절 알림 | 필수 |
| PUSH-04 | 긴급 정지 알림 | 필수 |
