# 연구 노트 (트레이딩 알고리즘 기획용)

이 폴더는 **연구원 에이전트 최박사**의 세부 연구 노트를 둘 수 있습니다.

- **최박사 메인 산출물**: 논문 형식의 통합 문서는 **`docs/트레이딩기법_연구자료.md`**입니다. 수집 자료 정리·출처 명시·관련 기법 연구·최신 트렌드·매매기법 가설·제안을 해당 문서에 계속 업데이트·추가하며, 작업 내용은 수시로 123에 기록합니다.
- **이 폴더**: 보조적으로 주제별 연구 노트(`연구_노트_주제_YYYYMMDD.md` 등)를 둘 때 사용합니다. 규칙: 인용·수치·사례에는 **정확한 출처** 첨부.
- **검증·결정·실전**은 사용자가 수행합니다.
