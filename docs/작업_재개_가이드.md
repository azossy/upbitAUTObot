## 3. 재개 시 체크리스트

### [기획관 / 사용자]
1. 123 열기 → "개발 완료 현황", "담당자별 이번에 할 일" 확인.
2. docs/기획_다음단계.md 등 기획서 갱신 여부 확인.

### [개발관]
1. 123 → "작업 지시 (개발관)"에서 [진행중] 항목 확인.
2. "■ 작업지시서"에 재작업 항목 있으면 해당 항목대로 수정.
3. 새 작업 지시가 있으면 코딩 → "최근 작업 내역" 기록, [완료!].

### [검증관]
1. 123 → "검증관 할당 작업"에 검증 요청 있는지 확인.
2. 요청 있으면 검증 수행 → 결과 갱신, [완료!]+날짜.
