## 4. 출시 검토 (부족·보완 제안)

### 4.1 v1.0에서 꼭 보완 권장

1. **서버 URL 설정** — 설정에서 API 서버 주소 입력·저장, ApiService가 해당 URL 사용.
2. **자동매매 책임 안내(디스클레이머)** — 봇 시작 근처 또는 설정 상단 고정 문구 1줄.
3. **계정 잠금 시 안내** — 403 시 "O분 후 다시 시도" 등 remaining_minutes 표시.

### 4.2 있으면 좋음 (v1.0 또는 v1.0.1)

- 이용약관·개인정보처리방침 링크
- CSV 파일 저장 옵션
- 봇 에러 한 줄 표시 (설정/대시보드)

### 4.3 v1.0 이후로 미룰 수 있는 항목

- JWT Refresh Token 앱 연동
- 거래내역 코인별 필터 (TRADE-03)
- 실시간 시세/캔들 차트 (UPBIT-05)
- iOS 지원

### 4.4 요약

| 구분 | 내용 |
|------|------|
| **v1.0 보완 권장** | 서버 URL 설정, 디스클레이머, 계정 잠금 안내 |
| **PRD 상 미완** | 실제 주문/취소(UPBIT-04)는 "검증용" 전제로 출시 가능. 실시간 시세·코인별 필터는 중기 권장 |
